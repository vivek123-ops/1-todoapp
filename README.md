
# MERN Todo App

A simple Todo Application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

## Features

* User Registration
* User Login
* Add Todo
* View Todos
* Delete Todo
* Authentication using Context API
* MongoDB Database Integration

## Tech Stack

### Frontend

* React.js
* Axios
* React Router DOM
* Bootstrap
* Context API

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose

## Installation

### Clone Repository

```bash
git clone <your-repository-url>
```

#### Run Frontend and Backend Together start

From the root folder:
npm install
npm run dev
This command starts both the frontend and backend servers simultaneously using Concurrently.
